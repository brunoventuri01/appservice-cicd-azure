Vantagens do CI/CD

O CI/CD permite automatizar o processo de integração, teste e entrega de aplicações, reduzindo falhas manuais e aumentando a velocidade das implantações. Com uma pipeline configurada, cada alteração enviada ao repositório pode ser publicada automaticamente no ambiente de produção ou homologação, garantindo mais rastreabilidade, padronização e segurança no ciclo de desenvolvimento.

Ambiente utilizado

Este projeto utiliza uma página HTML publicada em um Azure App Service, com o código-fonte armazenado em um repositório GitHub e deploy automatizado por meio do GitHub Actions.
